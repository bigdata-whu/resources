武汉大学计算机学院课程:
* 20202021725 《大数据 ―― 分析与处理》

当今世界，许许多多公司淹没在数据的海洋中，这是一个很好的模式。
但是，如何获取有用数据？如何基于大数据做出更好的决策？
这门课程将有助企业管理者学会如何将数据转化为竞争优势。

数据无处不在，其影响是无穷无尽的 ―――― 它可以帮助确定雇佣谁、如何设定价格、
关注哪个供应商以及该将营销资金放在何处。大数据、策略制订：
行为分析为在大数据时代做出更好决策和竞争的框架和工具、创新和见解。

通过本课程，我们将学习对大数据的基本理解，
学会如何利用横跨整个企业的数据来分析并创造性地进行战略决策。

#### 前置课程:


学分: 2.0

#### 主要教学内容:

课程侧重于五个关键领域，以提供更全面、更具创新性和可操作性的学习体验：

* 从概念框架和工具到设计思维、敏捷和数据可视化，数据驱动决策要点
* 体验式团队数据模拟项目，将学习付诸行动
* 通过数据分析（如营销、商业模式或人力资源）的实际应用，辅助与组织关联的可视化
* 对机器学习与人工智能在大数据领域取得的最新进展和未来发展进行独到分析和解读
* 了解使用大数据的风险、局限性和道德问题

主讲教师: 胡继承 教授
